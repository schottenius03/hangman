# 🕹️ Hangman

**Course project:** C++ game project  
**Description:** A Hangman game built using C++ with object-oriented programming and GUI development.

## 📌 Overview

Hangman is a graphical word-guessing game developed during a programming course.  
The project demonstrates object-oriented principles and GUI development in C++.  
Players guess letters to reveal a hidden word, with a visual interface showing progress and game state.

## 🛠️ Technologies Used

- C++ (object-oriented programming)  
- GUI development libraries (as used in project)  
- Git for version control

## 🎯 Key Takeaways

- Object-oriented programming in C++  
- GUI design and event handling  
- Applying Java OOP concepts to C++ implementation  
- Structuring a small game project for clarity and maintainability
